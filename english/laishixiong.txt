

中国人常犯的英语发音错误, 你中招了吗？
赖世雄
赖世雄
1
常春藤英语集团 创始人

我学习英语的过程分三个阶段：

一、矫正发音

二、会话

三、大量的阅与读

就第一个阶段而言, 是所有英语非母语人士的通病, 也是容易造成对学习没有兴趣的主因。

　　因为我初中时有日据时代留下来的本省籍老师，日本音很重，也有外省籍的老师，乡音也很重，没有一个真正可以用标准发音来教学进而引发学生学英语兴趣的老师。受到曹同学的激励，我开始很注重“说”来修饰门面，这个门面就是从发音开始。

　　英语音标总共有48个，包括有24个元音（元音）和24个辅音（子音）。成年人学英语时是按音标符号来学，但外国小朋友学英语是先看父母的嘴型，再模仿他们的口音，重复模仿，所以变成和父母一样的口音，任何母语都是这样学的。

　　但当时1966年，我没有发音标准又正确的音源，因此需要一台手提电唱器和一张外国人发音的唱片, 那时候的我已经懂得奋发图强，我开口向父亲借750元，而这笔钱是他薪水的两倍。父亲很久不说话，最后开口问我：“你是真的要好好学习吗？”我回答说：“这是真的，我现在知道了读书的重要性，我希望将来我的英语可以和曹同学一样好。”听了我的话，父亲二话不说，起身拿了大约800元新台币给我。于是我就去桃园市东方戏院旁边的电器行买了一台手提电唱机，还买了一张美国人灌的学习美式音标的黑胶唱片及一本教 K.K.（Kenyon and Knott 此两位作者名字的首字母）音标的小册子，准备要开始好好地学英语。从那天下午起，我就充分利用下课五分钟的时间、晚自习或其他零星的时间开始认真学音标。

我在听发音时把自己想象成是a newborn baby（一个新生儿），正在牙牙学语。从长元音 / i / 开始，接着短元音 / ɪ /，一次就只学两个音标。一天只学书里面的几个单词，例如 beat、bit、lead、lid，我就照着唱片的发音念 [ bit ]、[ bɪt ]、[ lid ]、[ lɪd ]。念完一遍，再念一遍，重重复复地练，像个新生儿一样少量地学音标，然后从下午到晚上不断地重复练习。在这过程中，我除了不停地念，我还每三分钟就听一下唱片，休息个一两分钟又继续听。

　　这种方式其实就是小孩子的学习方法，因为一般人能够专注的时间大概只有五分钟而已，所以我就利用这种五分钟的 attention span（专注时间）重复学习，很认真地模仿唱片中的发音。我特意不去理解这些练习发音英语单词的意思，因为小孩子还没有理解力，他们学英语是靠模仿的。 

　　我学发音的过程，就和小孩子刚开始讲“妈妈”的时候一样，可能会讲得不标准，也讲得不是很清楚，可是慢慢到了两三岁时，就能讲得标准了。所以我刚开始学音标时，发音绝对不可能和外国人完全一样。但我用成人拥有的学习能力，强迫自己把小孩子一年说话的量，压缩在一天内讲完，不断重复练习两个元音，我就可以讲好几千遍，相当于小孩要用上好几个月的时间学习，甚至是小孩子学一年以上才可能学达到的发音标准程度，我全都压缩在一天之内学习。到了晚上，我的发音就跟美国人一样讲得很相似了。

        例如，我在念girl这个单词时，不会念成 / gɝ /，而是念成 [ gɝl ]，类似中文的“各尔欧”，念到字尾的“欧”时，一定会把舌尖顶住上颚；我在念 world 这个单词时，不会念成 / wɝd / 或 / wəld /，而是念成 [ wɝld ]，类似中文的“沃尔欧的”， 但是尾音“的”几乎听不见。慢慢的也就知道有一些音是要切掉的，例如 “I wanted to do it.” 这句话中的 want，加 -ed 变成过去式或过去分词时，不能念成 [ `wantɪd ]，那个 / d / 是要切掉的，应念成 [ `wantɪ* ] 。 此外， put it out 这个词组要念成 [ ˌputɪ`tɑʊ * ]，这个 * 符号代表“停顿不发音”。 接着，我开始会发连音，例如 “Take it easy.”（别紧张 ， 慢慢来。）， 我会念成 [ ˌtekɪ`tizɪ ]。

        我在讲学与编辑书籍的过程中, 经常会有英语学习者向我提出如何才能学好英语发音的问题, 这些宝贵的问题经过整理, 也让我能明白英语非母语人士所遇上的各种疑问, 利用这次Live讲座的机会, 愿意与大家一同分享, 同时也邀请我的得意门生, 也是目前常春藤英语集团的教学总监Johnny吴纪维老师, 以他在英语教学上丰富的经验与一口流利地道的美式口音, 帮助大家一起战胜英语发音的种种问题, 把英语学得棒棒哒！
